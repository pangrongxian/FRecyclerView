##  如何使用？

# How to use ?

#### gradle

**Step 1.** Add the JitPack repository to your build file



	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}



**Step 2.** Add the dependency



	dependencies {
	        compile 'com.github.pangrongxian:FRecyclerView:1.0.2'
	}
